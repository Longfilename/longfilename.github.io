Below is a sampling of the projects and experiments housed in the longfilename repository.

Project: Short Documentation

Many moons ago, this project started as a ColdFusion application that would generate a project outline based on a JSON configuration file.

Currently it is a Yeoman generator that will generate a project outline based on a JSON configuration file, and documentation will be auo-generated based on the components and templates.

Experiment: Chord

A chord of a circle is a straight line segment whose endpoints both lie on the circle.

I ran across a JAR file online that contained this animation. I found it captivating, and recreated it in HTML/CSS/JS.
